All endpoints use query parameters (Vercel serverless compatible):

Auth:
  POST /api/auth                      (signup)
  POST /api/auth?action=login         (login)

Users:
  GET  /api/users?action=me           (current user)
  GET  /api/users                     (all users - admin)
  GET  /api/users?action=leaderboard  (leaderboard)
  PATCH /api/users?id={id}            (update role - superadmin)
  DELETE /api/users?id={id}           (delete - superadmin)

Projects:
  GET  /api/projects                  (list)
  GET  /api/projects?id={id}          (single)
  POST /api/projects                  (create)
  PUT  /api/projects?id={id}          (update)
  DELETE /api/projects?id={id}        (delete - admin)

Allowed Emails:
  GET  /api/allowed-emails            (list - superadmin)
  POST /api/allowed-emails            (add - superadmin)
  DELETE /api/allowed-emails?id={id}  (remove - superadmin)
